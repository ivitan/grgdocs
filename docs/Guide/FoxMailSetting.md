# Foxmail 邮箱客户端的配置使用

- 打开 Foxmail 邮箱客户端
 
- 点击新建账号

![](/imgs/FoxmailSetting/1.png)

- 点击手动设置

![](/imgs/FoxmailSetting/2.png)
 
- 在 POP 服务器里输入 **pop.grgbanking.com** SMTP 服务器里输入 **smtp.grgbanking.com**

![](/imgs/FoxmailSetting/3.png)
 
- 点击创建

![](/imgs/FoxmailSetting/4.png)
 
- 完成设置

![](/imgs/FoxmailSetting/4.png)
 
- 配置完成后的界面

![](/imgs/FoxmailSetting/6.png)
