# 修改邮箱密码

## 通过网页修改密码

- 通过浏览器访问 [https://mail.grgbanking.com](https://mail.grgbanking.com)

![](/imgs/ChangeMailPass/1.png)

- 输入账号密码登录

![](/imgs/ChangeMailPass/2.png)

- 点击左上角图标，点击后选择个人设置

![](/imgs/ChangeMailPass/3.png)

- 点击修改密码

![](/imgs/ChangeMailPass/4.png)

- 输入现用密码和新密码后选择保存更改即可

![](/imgs/ChangeMailPass/5.png)
