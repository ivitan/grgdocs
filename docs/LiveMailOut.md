# Windows Live Mail 邮件导出、导入
- 更换电脑，或者重装系统前，必须先做好邮件备份工作。如下是 Windows Live Mail中，邮件导出、导入教程。

## 在旧电脑或旧系统中导入邮件
> 在旧电脑或旧系统中，先把邮箱代理服务器中的邮件全部导出（存放位置要选择C盘以外的位置）。双击视频，可全屏播放
